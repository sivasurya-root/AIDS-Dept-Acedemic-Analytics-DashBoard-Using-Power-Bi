\# 🎓 AIDS Academic Insights Dashboard



\## 📌 Overview



This project presents an interactive \*\*Power BI dashboard\*\* designed to analyze student academic performance for \*\*B.Tech Artificial Intelligence and Data Science (AIDS)\*\*.



It transforms raw grade data into meaningful insights such as GPA, rankings, arrears, and subject-wise performance.



\---



\## 🚀 Features



\* 📊 GPA Calculation (Credit-based)

\* 🏆 Student Ranking System

\* ❌ Arrear Identification

\* 📈 Subject-wise Pass Percentage

\* 📊 Interactive Visualizations

\* 🎯 KPI Cards (Topper, Avg GPA, Total Students)



\---



\## 🛠️ Tech Stack



\* Power BI

\* DAX (Data Analysis Expressions)

\* Microsoft Excel (Data Source)



\---



\## 📂 Dataset



The dataset contains:



\* Student Names

\* Subject Grades

\* Credit Points
